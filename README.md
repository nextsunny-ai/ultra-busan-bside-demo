# B-SIDE RECORDS — 울트라백화점 부산 (시연용 데모)

써니엔터테인먼트 × 어반플레이 인터랙티브 SW 시연 데모.

## 시연 링크
- **메인**: https://nextsunny-ai.github.io/ultra-busan-bside-demo/

## 시연 조작
| 화면 | 키 |
|---|---|
| 인트로 | `ENTER` 또는 클릭 |
| 메인 LP | `←` `→` (트랙) / `M` (한 줄 쓰기) |
| 어플 4단계 | WELCOME → NICKNAME → WRITE → PREVIEW |
| 키오스크 PREVIEW | `P` 인쇄 |
| 인쇄 완료 | `SPACE` 다음 곡 |

## 콘텐츠
- 17곡 LP + 조그셔틀 (시연용 = 키보드 ←→)
- 모바일 어플 = 한 줄 이어쓰기
- B6 자켓 인쇄
